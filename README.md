## ENGLISH

### **Definitive Solution for Encrypted Wii U Games or Games Split into Multiple Files!**

Do you have an encrypted Wii U game with multiple `.h3` files and don't know what to do? Don't worry! Here's the quick and easy solution.

---

### **Quick Instructions**

1. **Open PowerShell in your game folder** (let's call it **folder X**).
2. **Run this command:**

   ```powershell
   Invoke-Expression (Invoke-WebRequest -Uri "https://raw.githubusercontent.com/CesarFRR/cdecrypt-script/main/decrypt.ps1" -UseBasicParsing)
   ```

3. **Make sure you have these files in your game folder:**
   - `title.tmd`
   - `title.tik`

   If you have these files, the script will do all the work for you and decrypt the game automatically.

---

### **Don't have these files? Don't worry!**

If you don't have the `title.tmd` and `title.tik` files, here's the solution:

1. Download the tool [WiiUDownloader](https://github.com/Xpl0itU/WiiUDownloader/releases/tag/v2.68).
2. Set the destination folder for the game you want to download.
3. **Start downloading the game.**  
   - **Option 1:** Wait 5 seconds, cancel the download, and check the destination folder. You'll find the `title.tmd` and `title.tik` files there.  
   - **Option 2:** Let WiiUDownloader download the entire game. This tool has an option to decrypt the game automatically once the download is complete.

4. Copy the `title.tmd` and `title.tik` files to your game folder (folder **X**).

---

### **Final Notes**

- This script uses `cdecrypt` to automatically decrypt Wii U games.
- If you encounter issues, check the [WiiUDownloader documentation](https://github.com/Xpl0itU/WiiUDownloader) or ask for help on Reddit forums.

Follow these steps and enjoy your decrypted Wii U games without hassle!
